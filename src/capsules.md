# Capsules

Capsules are small, immutable-ish “claims + evidence” bundles.

- **Claim**: a precise statement (even if provisional)
- **Context**: references, assumptions, scope
- **Artifacts**: JSON/MD, diagrams, hashes, optional proof stubs
